Python Data Analysis Projects
This repository contains a series of Python-based data analysis projects and exercises. The notebook explores various core programming concepts, data structures, and advanced numerical analysis using foundational libraries like NumPy, Pandas, and Matplotlib.

🚀 Key Projects & Analyses
1. Combinatorial Coin Problem
Goal: Determine the total number of ways to make 100 (or any given total) using various coin denominations (e.g., 100, 50, 25, 10, 5, 1).

Method: Implementation of nested loops and optimized range functions to calculate combinations.

Visualization: A plot showing how the number of combinations grows exponentially as the target total increases.

2. NumPy Numerical Computing
Mona Lisa Analysis:

Loading and processing image data stored as numerical values in a .txt file.

Utilizing N-dimensional arrays for fast memory processing of large datasets.

Visualizing the resulting numerical grid as an image using pp.imshow.

3. Core Python Data Structures
Lists & Slicing: Comprehensive exploration of list manipulation, including append(), extend(), insert(), and complex slicing syntax (e.g., squares[::-1] for reversing lists).

Dictionaries & Sets: Analysis of key-value pairs, using defaultdict for error handling, and sets for unique element tracking (e.g., world continents).

Comprehensions: Examples of efficient one-line creation of lists, dictionaries, and sets using if clauses and nested loops.

Data Classes: Comparison between using dictionaries, named tuples, and standard Python classes for structured data storage.

🛠️ Requirements & Dependencies
To run the notebook, you will need:

Python 3.x

NumPy: Fundamental package for numerical computing.

Pandas: High-performance data manipulation and analysis.

Matplotlib: Plotting and visualization library.
